Git Bash - Useful Lines

BASIC LINES

  • git init
  • git status
  • git add .
  • git commit -m "initial commit"
  • git push
  • git reflog
  • git branch (list local branches)
  • git checkout branch-name
  • git checkout -b "old-branch-name" "new-branch-name" (change current branch to another using the older branch and renaming it)
  • git merge
  • git pull (pull remote files to local folder)

LOCAL BRANCH LINES

  • git branch -m 'oldname' 'new name'
  • git branch --delete 'name'
  • git branch -d 'name'
  • git branch -a (list remote branches
  • git switch -c 'new-branch-name'
  • git pull origin master --allow-unrelated-histories

REMOTE LINES

  • git remote // to list remote name
  • git remote add
  • git remote rename
  • git remote remove
  • git remote set-head
  • git remote set-branches
  • git remote get-url
  • git remote rm (remove) origin
  • git remote add 'name' 'url'
  • git push 'remote name' --delete 'branch name'
  • git push 'remote name' -d 'branch name'
